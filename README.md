# Karnika Tailwind Landing Page

این پروژه یک پیاده‌سازی واکنش‌گرا از صفحه اصلی کارنیکا با HTML، Tailwind CSS و JavaScript است.

## اجرا
فایل `index.html` را مستقیماً در مرورگر باز کنید.

برای اجرای بهتر:
- افزونه Live Server در VS Code
- یا دستور زیر:
  python -m http.server 5500

سپس:
http://localhost:5500

## ساختار
- index.html
- js/app.js

## نکته
تصاویر فعلی نمونه و آنلاین هستند. برای نسخه نهایی، آدرس `src` تصاویر را با فایل‌های واقعی پروژه جایگزین کنید.
